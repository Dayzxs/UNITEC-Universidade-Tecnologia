Banco de Dados

Bancos de dados ou bases de dados são conjuntos de arquivos relacionados entre si, podendo conter registros sobre pessoas, lugares ou informações em geral

Ciencia da Computação

Ciência da computação é a ciência que estuda as técnicas, metodologias, instrumentos computacionais e aplicações tecnológicas, que informatizem os processos e desenvolvam soluções de processamento de dados de entrada e saída pautados no computador

Analise de Desenvolvimento de Sistemas

Análise de sistemas é a atividade que tem como finalidade a realização de estudos de processos a fim de encontrar o melhor caminho racional para que a informação possa ser processada. Um analista de sistemas pode trabalhar no projeto de criação de um novo sistema ou no serviço de manutenção de sistemas já prontos

Desenvolvimento Full Stack

O Desenvolvedor Full Stack é um programador com formação técnica completa para projetar, desenvolver, testar e distribuir web sites ou aplicativos. Ele é capaz de operar tanto no front-end quanto no back-end no desenvolvimento.

DevOps

O DevOps permite que funções anteriormente isoladas – desenvolvimento, operações de TI, engenharia da qualidade e segurança – atuem de forma coordenada e colaborativa para gerar produtos melhores e mais confiáveis.

Engenharia da Computação

A Engenharia de Computação é um curso que integra conhecimentos das áreas da Ciência da Computação e da Engenharia Eletrônica necessários para desenvolver hardware e software

Engenharia de Software

Atua nos processos de inovação, tanto no desenvolvimento de software como no de hardware. Na área de TI, é responsável principalmente pela automatização de processos e o desenvolvimento de soluções baseadas no processamento digital.

Gestão em E-commerce

Gestão de e-commerce é uma atividade que compreende o planejamento de um negócio online, bem como a execução e o monitoramento das atividades de tal empresa.

Jogos Digitais

A área de jogos digitais é um campo multidisciplinar que combina arte, tecnologia e narrativa para criar experiências interativas. Os profissionais dessa área desenvolvem jogos eletrônicos, explorando design, programação e animação para proporcionar entretenimento e engajar os jogadores.

Sistemas para Internet

Sistemas para internet são conjuntos de códigos e tecnologias que funcionam em conjunto para atender às necessidades específicas de aplicações online, como sites, plataformas de comércio eletrônico, sistemas de gerenciamento de conteúdo, aplicativos web e muito mais.

Redes de Computadores

Aprenda a planejar, implementar e fazer manutenção de redes de comunicação de dados no curso de Redes de Computadores da Estácio. Aqui, você aprende a configurar redes locais e de longa distância.
